Редактор XML базы данных - предметы.
Для работы требуется .Net версии не ниже 4.5.1
Загружает файл Items.xml, сохраняет в него же. Если файл не существует, создает новый.
Использование:
Открыть исполняемый файл.
Правая кнопка добавляет пустую строку.
Изменять поля в строке, поставив курсор на нужное поле. Редактирование начинается с нажатия любой клавиши, или даблклика мышью на поле.
По завершению редактирования программа сама сохраняет изменения.
Левая кнопка удаляет выделенную строку. Никаких предупреждений удаления.
Программа освобождает файл после закрытия.
Внимание!
Крайне не рекомендуется использовать файл items.xml каким либо образом, пока программа запущена!
Рекомендуется делать бекапы.
